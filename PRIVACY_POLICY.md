# Politique de Confidentialité - Akiprisaye

**Dernière mise à jour : Janvier 2026**

## Introduction

Akiprisaye est une application Android simple qui affiche le site web https://akiprisaye-web.pages.dev dans une vue web native (WebView). Cette politique de confidentialité explique notre engagement envers la protection de votre vie privée.

## Collecte de Données

**Nous ne collectons AUCUNE donnée personnelle.**

Cette application :
- Ne collecte pas d'informations personnelles identifiables
- N'utilise pas de services d'analyse (analytics)
- N'utilise pas de trackers publicitaires
- Ne partage aucune donnée avec des tiers
- Ne stocke pas d'informations localement en dehors du cache web standard

## Permissions

L'application demande les permissions suivantes :

### INTERNET
Nécessaire pour charger et afficher le contenu web de https://akiprisaye-web.pages.dev

### ACCESS_NETWORK_STATE
Utilisé uniquement pour vérifier la disponibilité de la connexion réseau

## Données Web

Lorsque vous utilisez l'application, vous accédez au site web https://akiprisaye-web.pages.dev. Les données échangées avec ce site web sont régies par la politique de confidentialité du site web lui-même, pas par cette application.

L'application utilise un WebView Android standard qui peut stocker :
- Cookies de session (si le site web en utilise)
- Cache web local (pour améliorer les performances)
- Historique de navigation local (effacé quand l'application est fermée)

Ces données restent locales sur votre appareil et ne sont jamais transmises à l'extérieur du WebView standard d'Android.

## Sécurité

- Toutes les communications avec le site web se font via HTTPS
- L'application n'a pas accès aux capteurs de votre appareil (caméra, microphone, GPS, etc.)
- L'application n'a pas accès à vos fichiers personnels
- L'application n'a pas accès à vos contacts ou autres applications

## Services Tiers

Cette application ne contient aucun SDK ou service tiers. Elle ne communique qu'avec le site web https://akiprisaye-web.pages.dev.

## Droits des Utilisateurs

Comme nous ne collectons aucune donnée, il n'y a aucune donnée personnelle à :
- Consulter
- Modifier
- Supprimer
- Exporter

Vous pouvez simplement désinstaller l'application à tout moment pour supprimer toutes les données locales (cache web).

## Modifications de cette Politique

Nous nous réservons le droit de mettre à jour cette politique de confidentialité. En cas de modification substantielle, une nouvelle version de l'application sera publiée avec une politique mise à jour.
